# 🚀 Smart Resource Allocation System

## 📌 Overview
The Smart Resource Allocation System is a data-driven platform designed to help NGOs and social organizations efficiently manage community needs. It transforms scattered and unstructured data into meaningful insights and prioritizes tasks based on urgency.

The system also suggests suitable volunteers, enabling faster and smarter decision-making.

---

## ❗ Problem Statement
Community organizations often collect important data through surveys and field reports. However:
- Data is scattered and unstructured  
- Difficult to identify high-priority needs  
- Inefficient volunteer-task allocation  

---

## 💡 Solution
This system provides:
- 📊 Structured data entry through a simple interface  
- ⚡ Priority scoring based on urgency  
- 🎨 Visual dashboard with color-coded urgency levels  
- 🤝 Volunteer matching based on skills and category  

---

## 🧠 Key Features
- Priority-based task ranking (High / Medium / Low)  
- Heatmap-style color visualization  
- Volunteer-task matching system  
- Simple and user-friendly interface  
- Lightweight and beginner-friendly  

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS  
- **Backend:** Python (Flask)  
- **Database:** SQLite  
- **IDE:** VS Code  

---

## 🌍 Impact
- Helps identify urgent needs quickly  
- Improves volunteer allocation efficiency  
- Enables data-driven decision-making  

---

## ▶️ How to Run
1. Clone the repository  
2. Open the project in VS Code  
3. Install dependencies: pip install flask
4. Run the app: python app.py
5. Open in Browser: http://127.0.0.1:5000/
